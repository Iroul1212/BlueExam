# BlueExam

**BlueExam** 是一款专为高校学生组织设计的 LaTeX 试卷/往年题整理模板。它旨在解决复习资料排版混乱、格式不统一的问题，支持一键切换“刷题版”和“解析版”，并内置了防盗水印与品牌 Logo 支持。

## ✨ 核心特性

* **🔄 一键双版本**：
    * **复习版**：包含蓝色答案填空、详细的蓝色解析框，适合内部核对与背诵。
    * **刷题版**：自动隐藏所有答案、填空留白、解析框消失，适合模拟考试。
* **🛡️ 底层防盗水印**：基于 `eso-pic` 的底层矩阵水印，覆盖全页，有效保护组织劳动成果。
* **🧩 智能题型支持**：
    * **填空题**：支持 `\tk{答案}`，短答案居中，长答案自动伸长下划线。
    * **选择题**：支持 `\xx` (4列)、`\xxtwo` (2列)、`\xxone` (单列) 三种排版模式。
* **🏷️ 品牌化定制**：轻松配置组织名称、Logo、课程信息与整理人。

## 🚀 快速开始 (Quick Start)

### 1. 环境要求
* **TeX 发行版**: TeX Live (推荐) 或 MiKTeX。
* **编译器**: 必须使用 **XeLaTeX** (用于支持中文)。
* **在线编辑**: 完美支持 [Overleaf](https://www.overleaf.com/)。

### 2. 使用方法
1.  下载本项目中的 `blueexam.cls` 和 `main.tex`。
2.  在 `main.tex` 中配置基本信息：
    ```latex
    \orgname{电气学院学生会学术部}
    \course{电路原理}
    \term{2024-2025学年}
    \setlogo{logo.png} % 可选
    ```
3.  **选择输出模式** (在 `main.tex` 头部)：
    ```latex
    \showanswerstrue  % 开启：导出带答案解析版
    % \showanswersfalse % 关闭：导出空白刷题版
    ```
4.  使用 **XeLaTeX** 编译即可。
