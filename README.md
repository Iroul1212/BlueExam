# BlueExam

**BlueExam** 是一款专为高校学生组织设计的 LaTeX 试卷/往年题整理模板。它旨在解决复习资料排版混乱、格式不统一的问题，支持一键切换“刷题版”和“解析版”，并内置了防盗水印与品牌 Logo 支持。

## ✨ 核心特性

* **🔄 一键双版本**：
    * **复习版**：包含蓝色答案填空、详细的蓝色解析框，适合内部核对与背诵。
    * **刷题版**：自动隐藏所有答案、填空留白、解析框消失，适合模拟考试。
* **🛡️ 底层防盗水印**：基于 `eso-pic` 的底层矩阵水印，覆盖全页，有效保护组织劳动成果。
* **🧩 智能题型支持**：
    * **填空题**：支持 `\tk{答案}`，短答案居中，长答案自动伸长下划线。
    * **选择题**：支持 `\xx` (4列)、`\xxtwo` (2列)、`\xxone` (单列) 三种排版模式。
* **🏷️ 品牌化定制**：轻松配置组织名称、Logo、课程信息与整理人。

## 🚀 快速开始 (Quick Start)

### 1. 环境要求
* **TeX 发行版**: TeX Live (推荐) 或 MiKTeX。
* **编译器**: 必须使用 **XeLaTeX** (用于支持中文)。
* **在线编辑**: 完美支持 [Overleaf](https://www.overleaf.com/)。

### 2. 使用方法
1.  下载本项目中的 `blueexam.cls` 和 `main.tex`。
2.  在 `main.tex` 中配置基本信息：
    ```latex
    \orgname{电气学院学生会学术部}
    \course{电路原理}
    \term{2024-2025学年}
    \setlogo{logo.png} % 可选
    ```
3.  **选择输出模式** (在 `main.tex` 头部)：
    ```latex
    \showanswerstrue  % 开启：导出带答案解析版
    % \showanswersfalse % 关闭：导出空白刷题版
    ```
4.  使用 **XeLaTeX** 编译即可。

# 🎉 Major Update: ElegantExam v2.0 发布

从 **BlueExam** 正式进化为 **ElegantExam**！本次更新专注于“颜值”与“自由度”，彻底解决了配色难、水印无法关闭等痛点。

## ✨ 新特性一览 (Highlights)

### 🎨 1. 内置 6 款设计师配色 (Designer Color Palettes)
我们发现手动调节 RGB 容易导致配色“辣眼睛”，因此 v2.0 锁定了 6 款符合学术审美的配色方案。无论你的校徽/Logo 是红色、绿色还是黄色，都能找到完美搭配：
* 🔵 **Blue (经典蓝)**
* 🔴 **Red (典雅红)**。
* 🟢 **Green (墨玉绿)**
* 🟣 **Purple (罗兰紫)**
* ⚫ **Black (极简黑)**
* 🟠 **Orange (活力橙)**

> **使用方法**：在 `main.tex` 中使用 `\settheme{blue}` (或其他颜色名) 即可一键换肤，解析框背景色会自动适配！

### 🛡️ 2. 全新的水印与声明控制 (Total Control)
现在你可以根据使用场景，自由开启或关闭页面元素：
* **水印开关**：`\showwatermarkfalse` —— 内部自用或打印时，保持页面清爽，节省墨水。
* **声明开关**：`\showdisclaimerfalse` —— 制作非公开发布版本时，隐藏黄色免责声明框。
* **底层修复**：改用 `eso-pic` 引擎，彻底修复了水印可能堆积在角落的 Bug，现在水印会完美平铺全页。

### 🛠️ 3. 排版功能增强
* **单栏选项**：新增 `\xxone` 命令，专治超长选项，强制一行一个，不再挤成一团。
* **智能填空**：`\tk` 命令现在能更智能地处理长答案的下划线延伸。
